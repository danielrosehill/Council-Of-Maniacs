# Business Ideas — Council of Maniacs

The user wants absurd business ideas generated by the Council of Maniacs. The input can be broader than a resume — any clues about the user work:
- Resume/CV or professional background
- Hobbies, interests, random skills
- A problem they've noticed in the world
- A professional website or portfolio URL
- Free-text description of anything relevant
- Any combination of the above

## Steps

1. **Gather clues**: Ask the user for their background/clues if not already provided. Be flexible — this mode accepts anything from a full resume to "I'm good at making spreadsheets and I hate mornings." Combine all sources into a single background string.

2. **Save background to a temp file**: Write the combined input to `data/career-input.txt` in the project root.

3. **Run the pipeline**: Execute from the project root:
   ```bash
   cd /home/daniel/repos/github/my-repos/Council-Of-Maniacs && uv run python -m backend.career.cli business-ideas -f data/career-input.txt
   ```

4. **Report results**: Tell the user where the PDF, Typst source, and JSON files were saved. Show the moderator's summary (the actionable takeaway).

5. **Offer follow-ups**: Ask if they want to also run `side-hustles` or `career-pivots` mode on the same input.

## Notes
- The pipeline calls OpenRouter with 8 personas in parallel, then peer review, then moderator synthesis — it takes a couple minutes.
- Output goes to `data/career-reports/` by default.
- This mode is the most flexible — users can provide minimal clues and the maniacs will run with them.
