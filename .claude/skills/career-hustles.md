# Career Hustles — Council of Maniacs

The user wants absurd side hustle ideas generated by the Council of Maniacs. You need their professional background — this can come from:
- A resume/CV file they point you to
- A professional website URL (fetch and extract)
- Free-text description of their background
- Any combination of the above

## Steps

1. **Gather background**: Ask the user for their professional background if not already provided. Accept file paths, URLs, or free text. Combine all sources into a single background string.

2. **Save background to a temp file**: Write the combined background to a temporary file at `data/career-input.txt` in the project root.

3. **Run the pipeline**: Execute from the project root:
   ```bash
   cd /home/daniel/repos/github/my-repos/Council-Of-Maniacs && uv run python -m backend.career.cli side-hustles -f data/career-input.txt
   ```

4. **Report results**: Tell the user where the PDF, Typst source, and JSON files were saved. Show the moderator's summary (the actionable takeaway).

5. **Offer follow-ups**: Ask if they want to also run `career-pivots` or `business-ideas` mode on the same background.

## Notes
- The pipeline calls OpenRouter with 8 personas in parallel, then peer review, then moderator synthesis — it takes a couple minutes.
- Output goes to `data/career-reports/` by default.
- If PDF compilation fails (missing fonts etc.), the Typst source is still saved and can be compiled manually.
